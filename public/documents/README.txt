Research Papers Directory
========================

Place your research paper PDFs here.

Current files should be:
- multi-agent-av-routing.pdf (Your AV routing paper)

The papers will be accessible at:
- https://your-domain.com/documents/filename.pdf

To add a paper:
1. Copy your PDF to this folder
2. Update the link in data/portfolio.ts
3. Rebuild or the dev server will auto-reload

See ADD_RESEARCH_PAPER.md in the root directory for detailed instructions.
